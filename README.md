# rainbowkit

## Library structure

```
use-ens (useENS)
connectors (useInjectedConnector, useWalletLinkConnector etc)
util (useConnector, useConnectOnMount, withWeb3React)
core (re-exports with custom tweaks)
badge (components for a Uniswap-like UI for wallet)
modal (auth modal w/ Rainbow design)
ui (re-exports of badge and modal)
```
