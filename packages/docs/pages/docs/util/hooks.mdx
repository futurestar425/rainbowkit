# Hooks

## Utility hooks

### useConnector

A React hook for using individual connectors from web3-react.

```tsx
import { Connector, ConnectorProps } from 'my-custom-connector'
import { useConnector } from '@rainbowkit/core'

export type CustomWalletOptions = ConnectorProps & { connectOnMount?: boolean }

export const useCustomConnector = (opts?: CustomWalletOptions) => {
  const options = opts || {}
  const { connectOnMount, ...injectedOptions } = options

  const injected = new InjectedConnector(injectedOptions)

  const ctx = useConnector(injected, connectOnMount)

  return ctx
}
```

## Connector hooks

### useInjectedConnector

A React hook to connect to a web3 wallet with web3-react `InjectedConnector`.
