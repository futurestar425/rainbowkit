# HOC

### withWeb3React

A HOC that wraps a component in a web3-react Provider.

```tsx
import React from 'react'
import { withWeb3React, useConnector } from 'rainbowkit/withWeb3Modal'

const App = () => {
  const { address, isConnected } = useConnector(someConnector)

  return <>{isConnected && address}</>
}

export default withWeb3React(App, (provider) => (provider.pollingInterval = 20000))
```
